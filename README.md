# 时光元 - 智能日程规划微信小程序

## 项目简介
时光元是一款将 **AI智能规划 + 游戏化激励 + 社交督促** 深度融合的大学生时间管理小程序。核心价值：让时间管理从"被动记录"变为"主动成长的数字自律伙伴"。

## 功能模块
1. **专注计时页** - 番茄钟计时器、任务管理
2. **数据统计页** - 可视化图表、成就系统
3. **AI智能规划页** - 四象限任务规划、AI拆解建议
4. **激励与奖励页** - 游戏化体系、抽卡系统、自定义奖励
5. **个性化设置页** - 主题切换、通知设置、数据管理

## 技术栈
- **前端**：微信小程序原生开发（WXML + WXSS + TypeScript）
- **后端**：Java Spring Boot（RESTful API）
- **数据库**：MySQL 8.0+
- **AI服务**：腾讯混元大模型API
- **部署**：微信云托管

## 快速开始

### 1. 环境准备
- 安装微信开发者工具
- 申请微信小程序AppID（或使用测试号）
- 安装Node.js（用于TypeScript编译）

### 2. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择`TODOLIST`目录
4. 输入AppID（或使用测试号）

### 3. 配置项目
1. 在`app.json`中配置页面路径
2. 在`project.config.json`中配置项目设置
3. 如需使用ECharts，需下载微信小程序版ECharts并放入`components/ec-canvas/`

### 4. 编译运行
1. 点击微信开发者工具的"编译"按钮
2. 预览小程序效果

## 核心特性实现

### 离线优先架构
- 所有写入操作在断网时自动缓存本地
- 网络恢复后静默同步到服务器
- 使用`utils/offline-cache.ts`管理离线队列

### 精准计时器
- 基于时间戳差值计算，解决切后台时间不准问题
- 支持暂停、继续、提前结束
- 使用`utils/pomodoro-timer.ts`提供核心计时功能

### AI智能规划
- 任务紧急度与重要性评估
- 四象限可视化展示
- 支持手动拖拽调整任务位置
- AI服务降级机制（网络异常时切换手动模式）

### 游戏化激励
- 积分和经验值系统
- 成就徽章收集
- 抽卡系统（单抽/十连抽）
- 自定义奖励系统

## 开发注意事项

### TypeScript配置
- 项目使用TypeScript开发
- 配置见`tsconfig.json`
- 需安装微信小程序类型声明：`miniprogram-api-typings`

### 网络请求封装
建议封装统一的网络请求工具：
- 自动添加认证token
- 统一错误处理
- 请求重试机制
- 离线队列管理

### 性能优化
1. **图片优化**：压缩图片，使用WebP格式
2. **代码分包**：按需加载，减少首次加载体积
3. **数据缓存**：合理使用本地存储
4. **渲染优化**：避免频繁setData

### 兼容性
- 微信基础库版本要求：2.19.4+
- 兼容iOS和Android
- 适配不同屏幕尺寸
